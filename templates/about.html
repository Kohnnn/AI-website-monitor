About AI Website Monitor

AI Website Monitor is a powerful tool designed to keep track of changes on your favorite websites. Using advanced AI technology, it can detect and analyze visual changes, providing you with timely notifications when something important happens.

Key Features

  • AI-Powered Analysis: Uses Google's Gemini Vision API to intelligently detect and describe changes
  • Custom Focus Areas: Tell the AI what aspects of websites matter most to you
  • Multiple Notification Options: Get alerts via email, Telegram, or Microsoft Teams
  • Flexible Scheduling: Set custom check intervals for each website
  • Visual History: Browse through a timeline of screenshots to see how websites evolve
  • Proxy Support: Monitor websites that might be region-restricted

How It Works

  1. Add websites you want to monitor
  2. Our system captures screenshots at your specified intervals
  3. AI compares new screenshots with previous ones
  4. If changes are detected, you receive notifications based on your preferences
  5. View detailed history and analysis of all changes

Technology Stack

AI Website Monitor is built using Python, Flask, SQLAlchemy, Redis Queue, and Playwright for browser automation. The AI analysis is powered by Google's Gemini Vision API.
